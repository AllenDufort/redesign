Problems with Old Design

Going on Heng's home page, I always have trouble trying to figure out how to use the website. In order to improve the website, I wrote down all the issues that I found relating to usability and accessability.

Usability

Usability is basically how easy it is for someone to use an interface, like a website. For Heng, I noticed a few usability issues in terms of efficiency, learnability, and memorability.

  • Learnability:
    • The function of the 3 circles on the right side is unclear
    • Unclear where to click to order at first glance
  • Memorability:
    • Hard to remember what tabs are under "More..."
    • Hard to remember where "Order" and "DoorDash" buttons are
  • Efficiency:
    • The user must scroll all the way down to find restaurant address

Accessibility

WebAIM is an online resource to analyze the accessibility of websites. Using WebAIM's analysis tools, I found a few accessibility issues with the website, which are listed below:

  • Low contrast between text and background
  • Low contrast between buttons and button text
  • Missing alt text for some web elements

Overall, I agree with the WebAIM analysis on the website's accessibility. However, there are 2 other points of inaccessibility that I found that WebAIM didn't mention:

  • 3 circles on the right-hand side have low contrast with the black background
  • Text too small to read on mobile devices
